# セキュリティポリシー

[English](SECURITY.md) | 日本語

## データプライバシー

この拡張機能は以下を満たします:

- **個人データを収集しません**
- **外部サーバーへデータを送信しません**
- **ワークスペース内のファイルをローカルで解析するだけです**
- **プロジェクトディレクトリ外のファイルへアクセスしません**

## ファイルアクセス

この拡張機能がアクセスする対象は以下のみです:

- ワークスペース（プロジェクト）ディレクトリ内のファイル
- 設定されたメディア拡張子に一致するファイル
- 参照検出のためのソースコードファイル

## セキュリティ上のベストプラクティス

- 拡張機能は VS Code のサンドボックス内で動作します
- ネットワークリクエストは行いません
- 実行時の外部依存はありません
- ソースコードは監査可能な形で公開されています
